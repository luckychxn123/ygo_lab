This project's theme is focused on yugioh cards mainly. 
- First page:
  lobby: allow users to create room, see all online users, watch commercial to earn game coins
  room of 2 players that allow users to chat, check each other's card decks and trade card
  room of multiple players to allow participants to chat, check the host's card decks and cards
- Second page:
  users can build and edit their deck with owned cards
- Third page:
  users can buy cards, there's limited card that all users can buy on upper section
